UI/UX Improvements to Make
🔴 Must Fix
1. No Visual Hierarchy on Answer Buttons
All 4 buttons are the same size and weight. The correct answer should feel like a "reveal moment" — animate it separately, bigger, with a checkmark slam.
2. Timer Has No Urgency Feel
Last 5 seconds should shake the whole screen slightly, turn the background red-tinted, and make the timer number huge and pulsing — not just a color change on a bar.
3. Leaderboard Is Just a List
No personality. Top 3 should look dramatically different — gold card, bigger font, crown icon, subtle glow border animation looping.
4. Host Screen Wastes Space
On desktop the question text and options use maybe 40% of the screen. The rest is empty. Use the full viewport — giant question text, huge answer blocks.
5. No Transition Between Game Phases
Question → Reveal → Next Question just snaps. Should slide or morph between phases with a dramatic wipe or zoom.

6. Player Waiting Screen Is Boring

The logo is just an emoji in a gradient box. A real wordmark with a custom icon makes it memorable.

When timer hits 5 seconds, add a full-screen vignette that pulses red at the edges (box-shadow inset on body). The timer number should grow from 2rem to 4rem. Add a heartbeat CSS animation on the entire answer grid. Play a faster, higher-pitched tick each second.
PIN Input:
Replace the single wide input with 6 individual digit boxes side by side (OTP-style). Each box highlights cyan when focused, auto-advances focus to next box on digit entry, shakes on wrong PIN, and glows green when all 6 filled. Keyboard backspace moves to previous box.
Phase Transitions:
Between Question → Reveal, animate the question card sliding UP and off screen while the reveal card slides UP from below. Between Reveal → Next Question, do a full-screen flash (white 80ms opacity pulse) then new question zooms in from scale 0.85.

All buttons: add transform: scale(0.94) on mousedown, release to scale(1) with spring easing. Player avatar emoji in status bar: bounce animation when score increases.
